自己的 blog 自己架 - Hexo X Github

GitHub 有提供 GitHub Pages 功能，讓使用者可以放靜態網站
利用編輯器(e.g., Brackets)編輯 Markdown 檔案
再利用 Hexo 來將 Markdown 檔案編譯成 html 並佈署到 GitHub 上
這樣 “自己的 blog 自己架” 的成就就達成啦!!
